## Estudos do projeto
- O objetivo era avaliar como subir o streamlit de maneira segura para os usuários. A versão inicial que encontrei foi com uso do GCP, subindo o aplicativo com containers docker e uso de duas ferramentas que eu não conhecia: App Engine e Cloud Run. O maior ponto agora é que isso irá gerar um custo que no momento não sei o quanto seria, e preciso verificar se essas são as melhoras e/ou únicas possibilidades.
